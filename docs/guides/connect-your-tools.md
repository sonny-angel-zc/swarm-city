# Connect your tools

Integrations turn Linear into your source of truth around product development. Keep data in sync and eliminate manual updates between tools.

![Connect your tools](https://uploads.linear.app/fe63b3e2-bf87-46c0-8784-cd7d639287c8/c2eae035-37e2-4754-adcb-b8305431aa1f/c92d70c7-e6d0-4fa2-a0fd-78f6e780993a)

## Key integrations

- [Slack](https://linear.app/settings/integrations/slack): create issues from Slack messages and sync threads.
- [GitHub](https://linear.app/settings/integrations/github) / [GitLab](https://linear.app/settings/integrations/gitlab): automate pull request and commit workflows, and keep issues synced both ways.
- [Agents](https://linear.app/integrations/agents): deploy AI agents that work alongside you as teammates.

## Browse all integrations

Discover 150+ available connections in the [Linear integration directory](https://linear.app/integrations), from bug creation via support tools (Intercom, Zendesk) to issues created from design explorations (Figma).

## Linear API

If you need something custom, build directly on the Linear API (GraphQL). See the [Linear developer docs](https://linear.app/developers).

## Completion checklist

- [ ] Connected at least one chat integration (Slack).
- [ ] Connected at least one code integration (GitHub or GitLab).
- [ ] Reviewed the integration directory for additional tooling.
- [ ] Confirmed API customization path via Linear developer docs.
